# zen-rs
zen in rust（rust 版缠论)

功能：
+ 多级别联立
+ 实时数据展示
+ 事件通知（系统桌面通知及飞书消息）
+ 支持 tradingview 的丰富画线工具
+ 支持多标的联立信号（如招商银行与上证指数叠加）
+ 行情回放
